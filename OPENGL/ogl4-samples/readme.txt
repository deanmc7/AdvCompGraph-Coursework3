OpenGL Samples Pack Usage
--------------------------------------------------------------------------------
The OpenGL Samples Pack's project can be generated for Visual C++ 2008 and 2010,
in 32 and 64 bits.

The sample run only when launched from Visual C++ IDE.

Linux, MacOSX or MinGW are not supported...
